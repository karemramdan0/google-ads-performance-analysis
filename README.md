# Google Ads Performance Analysis

## Project Overview
This project analyzes Google Ads campaign performance across 6 months (July–November).  
The goal is to evaluate profitability, customer behavior, and campaign efficiency.

---

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- Jupyter Notebook

---

## Key Insights
- High CTR does not guarantee profitability
- Coupon campaigns generate high traffic but low profit (price-sensitive users)
- Promo codes attract higher-intent buyers
- Limited offers (Black Friday) produce the highest conversion rates
- Mobile drives most traffic but not necessarily profit

---

## Project Structure
- `/data` → raw dataset  
- `/notebooks` → Python analysis  
- `/powerbi` → dashboard file  
- `/reports` → final report
